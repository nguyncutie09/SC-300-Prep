# Module 2

### Lab 12 - Manage Microsoft Entra smart lockout values

Configure the additional password protection settings for your organization.

##### Ex 1: Manage Microsoft Entra smart lockout values

1. Authentication methods > Password protection:
* Mục Custom smart lockout

Lockout threshold: số lần đăng nhập sai liên tiếp tối đa trước khi bị khóa tạm thời

Lockout duration in seconds: thời gian bị khóa tạm thời

NOTE: 

* Mục Custom banned passwords

Enforce custom list: Yes or no

Custom banned password list: nhập list ban, list nên có tên công ty bla bla

* Mục Password protection for Windows Server Active Directory

Enable password protection on Windows Server Active Directory: Yes or No

Mode: Enforced or Audit
