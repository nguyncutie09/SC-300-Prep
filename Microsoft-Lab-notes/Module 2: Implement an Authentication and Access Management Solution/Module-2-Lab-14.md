# Module 2

### Lab 14 - Enable sign in and user risk policies

##### Ex 1: Enable User risk policy

1. Cấu hình policy > vào ID Protection > Dashboard
2. Dưới mục Protect chọn User risk policy

Mục Assignments:

* Phần User, có thể Include và Exclude users
* Phần User risk, gồm High, Medium and above, Low and above

Mục Controls:

* Phần Access, có thể thực hiện Block hoặc Allow Access. Allow Access thì có thêm lựa chọn tick Require password change.

Tuy nhiên tháng 10,2026 phần Risk policy này sẽ migrate vào Conditional Access. nên từ giờ chỉ read-only được thôi.

3\. Dưới mục Protect chọn Sign-in risk policy

Tương tự với User risk, tuy nhiên phần Access thì Require MFA.
